# 🏦 О-Банк

Официальный цифровой сервис Центрального Банка Государства **О-Банк**.

## О проекте

Информационная система для обслуживания граждан:
- Подача заявок на кредиты
- Открытие счетов и вкладов
- Обмен валют
- Электронная очередь
- Цифровой Рубль
