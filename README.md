## 📋 О проекте

**О-Банк** — цифровая платформа Центрального Банка Государства для обслуживания граждан и юридических лиц в рамках ролевого сообщества.

Проект реализует полный цикл банковских услуг: от подачи заявки до получения готового договора с электронной подписью и печатью банка.

---

## ⚡ Функционал

### 🏛️ Для граждан
- 💳 Открытие расчётных и депозитных счетов
- 💰 Подача заявок на кредиты
- 💱 Обмен валют по актуальному курсу
- 📄 Электронные договоры с юридической силой
- 🎫 Электронная очередь на обслуживание
- 👤 Личный кабинет с историей операций

### 👨‍💼 Для сотрудников
- ✅ Одобрение/отклонение заявок
- 🪟 Управление окнами обслуживания
- 📊 Статистика работы
- ✍️ Электронная подпись документов
- 📞 Обработка обращений через Discord

### 🎩 Для директора
- 📈 Полная аналитика банка
- 💵 Управление курсами валют
- 👥 Управление персоналом
- 🔐 Эмиссия Цифрового Рубля
